# Crime Data Analysis using SQL

This project focuses on analyzing crime-related data using SQL queries. 
The dataset contains information about offenders, victims, crime categories, 
and report types.

## Objectives
- Retrieve and filter crime data using SQL queries
- Perform data analysis using aggregate functions
- Understand grouping and sorting of records
- Practice SQL operations such as INSERT, DELETE, and JOIN

## SQL Concepts Used
- SELECT statements
- WHERE filtering
- COUNT, AVG, and SUM aggregate functions
- GROUP BY and HAVING clauses
- ORDER BY and LIMIT
- Subqueries
- INSERT and DELETE operations
- JOIN operations
- Pattern matching using LIKE

## Tools Used
- SQL
- PostgreSQL / SQLite
- GitHub for project version control

## Project Files
- `queries.sql` – Contains all SQL queries used for data analysis
- `project_report.pdf` – Detailed explanation of the project and queries
